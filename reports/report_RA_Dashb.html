Project Plan

Develop a web-based Revenue Assurance Dashboard for a telecom finance team to ingest daily CDR and billing feeds, reconcile data to detect revenue leakage, and provide drill-down reporting with KPIs. The system must support role-based access and handle large daily data volumes efficiently, ensuring secure and accurate financial insights.

Total effort: 80hRecommended team: 3 peoplePhases: 2

Technology Stack

BackendFastAPI + Python 3.11
FrontendReact 18 + TypeScript
DatabasePostgreSQL 15 + Redis for caching
InfrastructureDocker + AWS ECS
Key Librariespandas, SQLAlchemy, react-table, redis-py, auth0
